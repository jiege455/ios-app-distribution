# iOS/Android 应用分发平台

一个简洁、美观的iOS和Android应用分发平台，支持IPA和APK文件的上传、解析和分发。

## 功能特点

- 🍎 **iOS应用分发** - 支持IPA文件上传，自动生成安装清单
- 🤖 **Android应用分发** - 支持APK文件上传，自动解析应用信息
- 📱 **响应式设计** - 完美适配手机、平板和电脑
- 🎨 **现代UI** - 简洁美观的用户界面
- 🔐 **安全防护** - 密码保护、频率限制、SQL注入防护
- 📊 **后台管理** - 应用管理、系统设置、操作日志
- ☁️ **多云存储** - 支持本地、七牛云、阿里云OSS、腾讯云COS、FTP
- ⏰ **定时清理** - 支持宝塔面板定时任务清理

## 环境要求

- PHP >= 7.4
- MySQL >= 5.6
- GD扩展（用于图片处理）
- ZipArchive扩展（用于APK解析）

## 安装步骤

1. 下载源码并上传到服务器
2. 访问网站根目录，自动跳转到安装页面
3. 填写数据库信息和管理员账号
4. 完成安装

## 默认账号

- 用户名: `admin`
- 密码: 安装时设置

## 目录结构

```
├── admin/              # 后台管理
├── includes/           # 核心文件
├── uploads/            # 上传目录
├── manifests/          # iOS安装清单
├── qrcode/             # 二维码生成
├── php-apk-parser/     # APK解析库
├── index.php           # 前端首页
├── api.php             # API接口
├── install.php         # 安装程序
└── cleanup.php         # 定时清理
```

## 安全建议

1. 修改默认管理员密码
2. 设置复杂的密钥（secret_key）
3. 启用HTTPS
4. 定期备份数据库
