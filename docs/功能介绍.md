# iOS应用分发平台 - 系统功能介绍

## 系统概述

iOS应用分发平台是一款专业的移动应用分发管理系统，支持iOS和Android应用的在线分发、安装和管理。系统采用PHP + MySQL开发，界面美观、功能完善、易于部署。

---

## 核心功能

### 📱 应用管理

| 功能 | 说明 |
|------|------|
| 应用上传 | 支持 IPA 和 APK 格式，自动解析应用信息 |
| 应用解析 | 自动提取应用名称、版本号、Bundle ID、图标 |
| 密码保护 | 支持设置访问密码，保护应用安全 |
| 应用分组 | 支持应用分类管理 |
| 批量管理 | 批量删除、批量移动分组 |
| 应用编辑 | 修改应用名称、版本、描述等信息 |

### 📤 上传功能

| 功能 | 说明 |
|------|------|
| 拖拽上传 | 支持拖拽文件到上传区域 |
| 进度显示 | 实时显示上传进度和速度 |
| 大文件支持 | 支持上传500MB以上大文件 |
| 文件验证 | 自动验证文件类型和格式 |
| 断点续传 | 网络中断后可继续上传 |

### 📥 下载安装

| 功能 | 说明 |
|------|------|
| iOS安装 | 生成plist文件，支持itms-services协议安装 |
| Android安装 | 直接下载APK文件安装 |
| 二维码下载 | 每个应用生成独立二维码 |
| 密码验证 | 设置密码的应用需要验证后才能下载 |

### ☁️ 存储支持

| 存储类型 | 说明 |
|---------|------|
| 本地存储 | 默认存储方式，文件保存在服务器本地 |
| 七牛云 | 支持七牛云对象存储 |
| 阿里云OSS | 支持阿里云对象存储 |
| 腾讯云COS | 支持腾讯云对象存储 |
| FTP | 支持FTP远程存储 |

### 🎨 前台功能

| 功能 | 说明 |
|------|------|
| 应用列表 | 展示所有应用，支持分页显示 |
| 应用搜索 | 按名称搜索应用 |
| 应用详情 | 显示应用图标、名称、版本、大小等信息 |
| 公告系统 | 支持发布网站公告 |
| 广告位 | 支持配置首页和安装页广告 |

### ⚙️ 后台管理

| 功能 | 说明 |
|------|------|
| 仪表盘 | 数据统计、系统概览 |
| 应用管理 | 查看、编辑、删除应用 |
| 举报处理 | 处理用户举报信息 |
| 公告管理 | 发布和管理公告 |
| 广告管理 | 配置广告位内容 |
| 操作日志 | 记录所有操作行为 |
| 系统设置 | 配置网站参数 |
| 系统更新 | 在线检测和安装更新 |

### 🔒 安全功能

| 功能 | 说明 |
|------|------|
| 文件类型验证 | 验证文件真实类型，防止伪装攻击 |
| 上传频率限制 | 防止恶意上传攻击 |
| 密码保护 | 应用访问密码保护 |
| 操作日志 | 记录所有操作，便于追溯 |
| 维护模式 | 一键开启维护模式 |

### ⏰ 定时任务

| 功能 | 说明 |
|------|------|
| 过期清理 | 自动清理过期应用 |
| 日志清理 | 自动清理过期日志 |
| 统计清理 | 自动清理过期统计数据 |

## 系统要求

### 服务器环境

| 环境 | 最低要求 | 推荐配置 |

## 安装部署

### 快速安装

1. 上传代码到服务器
2. 访问 `http://域名/setup.php`
3. 按向导完成安装
4. 登录后台配置系统

### 宝塔面板部署

详细教程请参考：`docs/宝塔搭建教程.md`

## 更新升级

### 在线更新

1. 后台点击「系统更新」
2. 点击「检查更新」
3. 发现新版本后点击「一键更新」
4. 系统自动备份并更新

### 手动更新

1. 下载最新版本
2. 解压覆盖文件（排除data、uploads目录）
3. 访问网站检查是否正常
