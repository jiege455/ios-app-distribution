# iOS应用分发平台 - 宝塔面板搭建教程

## 一、环境要求

| 环境 | 要求 |
|------|------|
| PHP版本 | 7.4 及以上 |
| MySQL版本 | 5.6 及以上 |
| Nginx/Apache | 任选其一 |
| 扩展 | fileinfo、zip、curl、mysqli |

---

## 二、宝塔面板安装

### 2.1 安装宝塔面板（如已安装可跳过）

**CentOS安装命令：**
```bash
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh
```

**Ubuntu/Debian安装命令：**
```bash
wget -O install.sh http://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh
```

安装完成后会显示面板地址和登录信息，请妥善保存。

---

## 三、安装运行环境

### 3.1 登录宝塔面板

浏览器访问面板地址，输入用户名密码登录。

### 3.2 安装软件

在【软件商店】中安装以下软件：

| 软件 | 推荐版本 | 说明 |
|------|---------|------|
| Nginx | 1.20+ | Web服务器 |
| PHP | 7.4 或 8.0 | 脚本语言 |
| MySQL | 5.7 或 8.0 | 数据库 |
| phpMyAdmin | 最新版 | 数据库管理（可选） |

### 3.3 安装PHP扩展

点击【软件商店】→【PHP】→【设置】→【安装扩展】，安装以下扩展：

- fileinfo（必须）
- zip（必须）
- curl（必须）
- mysqli（必须）
- opcache（推荐，性能优化）
- redis（可选，缓存加速）

### 3.4 修改PHP配置

点击【软件商店】→【PHP】→【设置】→【配置修改】：

```
upload_max_filesize = 500M      ; 最大上传大小
post_max_size = 500M            ; POST数据大小
max_execution_time = 600        ; 脚本执行时间
max_input_time = 600            ; 输入时间
memory_limit = 256M             ; 内存限制
```

修改后点击【保存】并【重载配置】。

## 四、创建网站

### 4.1 添加站点

点击【网站】→【添加站点】：

| 设置项 | 填写内容 |
|--------|---------|
| 域名 | 您的域名（如：app.example.com） |
| 根目录 | /www/wwwroot/app.example.com |
| FTP | 不创建 |
| 数据库 | MySQL（选择utf8mb4编码） |
| PHP版本 | PHP-74（或您安装的版本） |

点击【提交】创建站点。

### 4.2 记录数据库信息

创建成功后会显示数据库信息，请记录：

```
数据库名：app_example_com
用户名：app_example_com
密码：xxxxxxxxxx
```

## 五、上传代码

### 5.1 方式一：本地上传

1. 点击【文件】→ 进入网站根目录
2. 删除默认生成的文件（index.html、404.html等）
3. 点击【上传】→ 选择本地代码压缩包
4. 右键压缩包 → 【解压】

### 5.2 方式二：远程下载

1. 点击【文件】→ 进入网站根目录
2. 点击【远程下载】
3. 输入下载地址（如GitHub Release地址）
4. 下载完成后解压

### 5.3 设置目录权限

选中以下目录，右键→【权限】→ 设置为 755 或 777：

```
uploads/          （上传目录）
data/             （数据目录）
manifests/        （plist文件目录）
qrcode/           （二维码目录）
error.log         （日志文件，如不存在会自动创建）
```

## 十、常见问题

### Q1: 上传文件失败？

**解决方案：**
1. 检查PHP配置中的 upload_max_filesize 和 post_max_size
2. 检查 uploads 目录权限是否为 755 或 777
3. 检查磁盘空间是否充足

### Q2: 页面显示空白？

**解决方案：**
1. 检查PHP版本是否 >= 7.4
2. 检查是否安装了必要的PHP扩展
3. 查看 error.log 日志文件排查错误

### Q3: 数据库连接失败？

**解决方案：**
1. 确认数据库服务已启动
2. 检查数据库用户名密码是否正确
3. 检查数据库是否已创建

### Q4: iOS应用无法安装？

**解决方案：**
1. 确保已配置SSL证书（iOS要求HTTPS）
2. 检查plist文件是否正常生成
3. 检查应用签名是否有效

### Q5: 如何修改上传大小限制？

**解决方案：**
1. 宝塔面板 → 软件商店 → PHP → 设置 → 配置修改
2. 修改 upload_max_filesize 和 post_max_size
3. 保存并重载配置

### Q6: 如何备份数据？

**解决方案：**
1. 备份数据库：宝塔面板 → 数据库 → 备份
2. 备份文件：宝塔面板 → 文件 → 压缩以下目录
   - uploads/
   - data/
   - manifests/

## 十二、安全建议

1. **修改后台入口**
   - 将 admin.php 重命名为其他名称

2. **设置目录保护**
   - 禁止访问 data/ 目录
   - 在Nginx配置中添加：
   ```nginx
   location ~* ^/(data|includes)/ {
       deny all;
   }
   ```

3. **定期备份**

   - 备份数据库和上传文件
